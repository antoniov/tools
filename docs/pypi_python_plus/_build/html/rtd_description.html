Python supplemental features¶

python_plus adds various features to python 2 and python 3 programs. It is designed to be used as integration of pypi future to help to port your code from Python 2 to Python 3 and still have it run on Python 2.

vem: virtual environment manager¶

This packge is released with an nice command: vem that is an interactive tool with some nice features to manage standard virtual environment and it is osx/darwin compatible.


This module is part of tools project.
