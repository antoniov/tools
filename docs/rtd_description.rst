Python & bash tools
-------------------

Sparse python and bash source code.

These tools help to cover following areas of software:

* Odoo deployment
* Odoo database maintenance (creation and upgrade, massive)
* Odoo database profiling (auto)
* Database check (auto & massive)
* Development
* Documentation
* Testing
